HTML Tags

HTML Tag Breakdown

Tag Name Description
<div> Tag A generic container for grouping elements.
<span> Span A generic inline container for text or other inline elements.
<header> Header Defines a header section for a webpage or a section within the page.
<body> Body Defines a body section for a webpage or a section.
<footer> Footer Defines a footer section for a webpage or a section.
<section> Section Represents a distinct section of content within a document.
<article> Article Represents a self-contained piece of content.
